Information

Have you ever noticed that your diffuser liquid stops decreasing and the scent no longer spreads? The sticks (reed sticks) used in diffusers have a lifespan. It is generally recommended to replace them once a month.

The DiffuserStick app lets you record the date when you inserted the diffuser sticks along with a photo. When itâ€™s time to replace them, you will receive a push notification reminder.

How to Use

  • List Tab: Tap the [+] button at the top right to add a diffuser. Use the button at the top left to sort the list. Added diffusers will appear in the list.
  • Archive Tab: Diffusers you no longer use can be stored using the [Archive] feature.
  • Default Period: The default replacement cycle is set to 30 days. You can change it in 15-day increments.
  • Add Diffuser: Set the title, photo (permission required), memo, and installation date. When the replacement period is reached, a push notification (permission required) will notify you.
  • Diffuser Details: Tap a diffuser in the list to view detailed information.
    • Tap on the image to view it full screen and enlarge it.
    • After replacing the sticks, tap the [Replace] button to update the replacement date to today.
    • You can also use [Archive], [Edit], and [Delete] functions.
    • The [Share] icon allows you to save the image or send the diffuser contents to an external service.
  • In-App Purchase: You can purchase a product that removes all advertisements from the app. If you need to restore purchases (e.g., after reinstalling), tap the [Restore Purchases] button.

Additional Features

  • Add Widget (iOS 17 or later): Add a widget to your Home Screen to quickly check diffuser status. Tapping the widget opens the corresponding diffuser in the app. Supports Liquid Glass theme and full-color image selection in [Edit Widget].
  • Rewrite from Archive: Tap [Rewrite] in the archive to create a new diffuser based on archived content.
  • App Theme Selection (iOS 13 or later): Choose between Light Mode, Dark Mode, or follow the system setting.
